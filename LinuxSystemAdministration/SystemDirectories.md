'/' &rarr; root directory

home / '~' &rarr; home directory

bin &rarr; binaries, executable programs

etc &rarr; configuration files

media &rarr; removeable media (cdrom, etc...)

var &rarr; files that change often (logs, databases)

root &rarr; home folder for root user

usr &rarr; user programs and files 

boot &rarr; files needed to start the computer

dev &rarr; devices

lib &rarr; essential code shared by programs

mnt &rarr; mounting points

opt &rarr; third party softwares

proc &rarr; virtual files showing system/CPU info

temp &rarr; temporary files that deleted after the reboot
